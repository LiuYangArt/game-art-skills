# Unreal Custom Node Skill

这个 Skill 用来帮你把一句效果描述，直接变成 Unreal 材质里可复制使用的内容。

## 你可以怎么说

- “帮我写一个海洋的 Custom Node”
- “我要一个扫描线 CRT 效果”
- “给我一个溶解边缘效果，接到 Emissive”
- “帮我输出完整节点组，我想直接粘贴到材质编辑器”

## 它会给你什么

- 一份可以直接复制到 `Custom Node -> Code` 里的 HLSL
- 一份对应的 Custom Node 蓝图节点代码
- 如果你要求“完整节点组”，还会额外给你整组可粘贴节点
- 如果需要，还可以帮你做静态验证，减少报错

## 你描述时最好带上这些信息

- 想要什么效果
- 最终接到哪里：`Base Color`、`Emissive`、`Normal`、`WPO` 等
- 有没有纹理输入
- 是屏幕空间、UV 空间，还是世界空间效果
- 想要偏真实，还是偏便宜快速

## 最简单的说法

你不需要懂 HLSL，也不需要先搭 pin。

直接说：

“我要一个 XXX 效果，输出完整节点组。”

或者：

“我要一个 XXX 的 Custom Node，只给我 HLSL 和对应 pin。”

## 小提示

- Unreal 默认是 `Z` 轴朝上
- 世界单位默认是厘米
- Custom Node 新建输入时，通常只需要填名字，类型由外部连线决定
